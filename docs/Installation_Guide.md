# Installtion Guide

This page covers the process of installation

## Installation

First the install script will be found [here](https://github.com/AbeonaSec/abeonasec-core/blob/main/install.sh)

Upon running it there will be several steps it runs through;

- First you will be prompted to update your keyring
  - This can be declined however it is necessary to install the software
- Next it will check to see if any of the needed dependencies are already installed, then install any/all missing
  - Once again, this can be declined but they are necessary for functionality
- Finally it will install the Nvidia Container Toolkit

Once this has finished you will be prompted to reinstall your system for the changes to take affect.
